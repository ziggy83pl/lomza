Instrukcja — własne miasto w 4 krokach

Krok 1 — Znajdź TERYT swojego miasta

https://bdl.stat.gov.pl/api/v1/units?name=NazwaMiasta&lang=pl&format=json
Zwraca listę jednostek z ich identyfikatorami. Znajdź id dla swojego miasta.

Krok 2 — Pobierz dane dla wskaźnika

https://bdl.stat.gov.pl/api/v1/data/by-variable/72305?unit-id=TWOJ_TERYT&format=json
Zastąp 72305 ID wskaźnika (np. populacja), a TWOJ_TERYT kodem TERYT.

Krok 3 — Zaktualizuj stałe danych w pliku

const CITY_DATA = {
  name: "Twoje Miasto",
  teryt: "XXXXXXX",
  population: XXXXX,
  budget_income: XXXX,
  ...
};
